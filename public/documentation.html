MyFlix API

The objective of this project is to build the server-side component
of a movie web application that will let users access information about
different movies, directors, and genres. Users will also be able to sign
up, update their personal information, and create a list of their
favorite movies.

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format Example Request Example Response
Return a list of all movies /movies GET None A JSON object holding data about all the movies /movies {

Return data about a single movie by title /movies/:title GET None A JSON object holding data about a single movie, containing title, director, genre, description, image URL, whether it is featured or not /movies/Love%20Actually {
"title": "Love Actually",
"director": "Richard Curtis",
"genre": "Drama, Comedy, Romance",
"description": "The movie follows the stories of different people dealing with their love lives",
"imageURL": "love-actually.png",
"isFeatured": "true"
}
Return data about a genre by title /movies/genres/:title GET None A JSON object holding data about a movie genre, containing the name and a description /movies/genres/romance {
"title": "Romance",
"description": "The romance genre tells stories about romantic relationships"
}
Return data about a director by name /movies/directors/:name GET None A JSON object holding data about a director, containing name, bio, birth year, death year /movies/directors/Nora%20Ephron {
